# **App Name**: Sobhy Digital Showcase

## Core Features:

- Hero & Introduction: Prominently display Ahmed Sobhy's name, professional title, a compelling tagline, and a high-quality professional headshot.
- About Me Section: A dedicated area to present Ahmed's professional background, career journey, and a concise overview of his key skills and expertise.
- Project Showcase: A dynamic grid or carousel display of past projects, each featuring an engaging image thumbnail, title, a brief description, and external links to live demos or code repositories.
- Contact Information Display: Clearly present multiple ways to contact Ahmed, including email address, LinkedIn profile, GitHub, and any other relevant social or professional platforms.
- Responsive Design: Ensure the entire portfolio website automatically adjusts its layout and content to provide an optimal viewing and interaction experience across all devices, from mobile phones to large desktop screens.
- AI-Powered Description Tool: An internal tool that helps Ahmed generate engaging and descriptive content or taglines for his portfolio projects and about section, based on provided keywords or prompts, saving him time and enhancing impact.

## Style Guidelines:

- Color scheme: A sophisticated dark mode, emphasizing professionalism and modern aesthetics.
- Primary accent color: A vibrant, clear blue (#4092E1), signifying trust, technology, and clarity. This hue (HSL: 210, 75%, 60%) provides excellent contrast against a dark background.
- Background color: A very dark, desaturated blue-gray (#1F2630). This deep tone (HSL: 210, 25%, 12%) ensures text readability and complements the primary accent.
- Secondary accent color: A luminous cyan (#85EDF1) with a distinct hue (HSL: 180, 80%, 70%) to create visual interest and highlight interactive elements without clashing with the primary blue.
- Headline and body text font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance across all screen sizes, suitable for both concise statements and detailed project descriptions.
- Use clean, minimalist line icons for social media links, contact methods, and section markers to maintain a sleek and contemporary professional look.
- Implement a modular, section-based layout with generous dark space, ensuring clear visual hierarchy and an uncluttered user experience, similar to premium digital agency portfolios.
- Incorporate subtle micro-interactions such as gentle hover effects on portfolio cards, smooth scrolling transitions between sections, and tasteful fades or slides for content loading, enhancing engagement without distracting from the content.
